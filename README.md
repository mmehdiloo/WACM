# WACM PROJECT
# Empirical illustration: US banking data (1986)

This repository contains the GAMS code and data required to reproduce the empirical illustration in Mehdiloo & Kerstens (2026).

## Data source

We utilize a secondary data set of US banking data initially used in Aly, Grabowski, Pasurka, and Rangan (1990).  
There are **322 observations** for the year 1986, with:

- 3 inputs  
- 3 input prices  
- 5 outputs  

## Repository contents

| File | Description |
|------|-------------|
| `bankdata1990.xls` | Data (322 observations) |
| `main.gms` | Main GAMS code for estimation |
| `mincost.txt` | Output file generated by GAMS |
| `mincost.xlsx` | Excel file with summary statistics (sum, min, max, mean) created from `mincost.txt` |

## Requirements

- **GAMS** (version 24.6.1 or higher)  
- Any GAMS license that supports LP/MIP

## Workflow

### Step 0: Data setup
Copy the data file `bankdata1990.xls` to your C: drive.

(Note: Copy directly to the drive, not to any folder.)

### Step 1: Run GAMS
1. Open GAMS IDE or GAMS Studio
2. Load `main.gms`
3. Run the model (F9)
4. Results are exported to `mincost.txt`

### Step 2: Post-process in Excel
1. Import the data from `mincost.txt` into `mincost.xlsx`
2. Compute summary statistics (sum, min, max, mean, etc.)

## Expected outputs

The code reproduces:
- Main tables and figures in the empirical illustration section  
- Cost efficiency scores and summary statistics
